Architektur

Front-End Processor:

Benutzerinterface mit Authentication, Entgegennahme der Belege. Nach Rechnungstrennung werden die Belege an das Logisth.AI Backend zur Bearbeitung weitergeleitet. Das Frontend wird über den Fortschritt oder gegebenenfalls Fehler informiert

Logisth.AI Backend Engine

Verarbeitet die übergebenen Belegdateien vom OCR Prozess bis zur Erstellung der Ergebnisse. Kommuniziert mit dem Schnittstellenserver auf Kundenseite. Die Ergebnisse werden in ein Queuing System für den Import gestellt

Schnittstellen Server

Dieser ist auf Kundenseite in dessen Infrastruktur installiert. Stellt der Logisth.AI Backend Engine und dem Front-End Prozessor Informationen zur Laufzeit zur Verfügung. So kann zu einer eingegebenen Kundennummer am Frontend ein Firmenname mit UID Nummer angezeigt werden oder das Backend Beleginformationen mit dem jeweiligen Kunden/Lieferantenstamm vergleichen.

Import Service

Übernimmt vom der Backend Engine die Resultate und übermittelt diese in die Buchhaltungssysteme
